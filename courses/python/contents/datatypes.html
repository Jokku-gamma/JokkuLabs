Python Data Types

In this lesson, we'll explore the fundamental building blocks of data in Python: Data Types. Understanding these types is crucial for writing effective and error-free Python code, as they dictate what kind of values variables can hold and what operations can be performed on them.

What are Data Types?

A data type is a classification that specifies which type of value a variable has and what type of mathematical, relational, or logical operations can be applied to it without causing an error. Python is a dynamically typed language, meaning you don't need to explicitly declare the data type of a variable when you create it; Python infers it at runtime.

Common Python Data Types

1. Numeric Types: `int`, `float`, `complex`

These represent numbers.

  • `int` (Integer): Whole numbers, positive or negative, without a decimal point.
  • `float` (Floating-point number): Numbers with a decimal point, or numbers written using scientific notation.
  • `complex` (Complex number): Numbers with a real and imaginary part (e.g., `3 + 5j`).

# Integers
age = 30
quantity = -10

# Floats
price = 19.99
pi = 3.14159

# Complex numbers
z = 2 + 3j

print(f"Age: {age}, Type: {type(age)}")
print(f"Price: {price}, Type: {type(price)}")
print(f"Complex Number: {z}, Type: {type(z)}")
Expected Output: 
Age: 30, Type: 
Price: 19.99, Type: 
Complex Number: (2+3j), Type:

2. Text Type: `str` (String)

Strings are sequences of characters, used for text. They are enclosed in single quotes (`'...'`), double quotes (`"..."`), or triple quotes (`'''...'''` or `"""..."""`) for multi-line strings.


name = "Joel"
greeting = 'Hello, Jokkulabs!'
multiline_text = """
This is a
multi-line string.
"""

print(f"Name: {name}, Type: {type(name)}")
print(f"Greeting: {greeting}, Type: {type(greeting)}")
print(f"Multiline Text: {multiline_text}, Type: {type(multiline_text)}")
Expected Output: 
Name: Joel, Type: 
Greeting: Hello, Jokkulabs!, Type: 
Multiline Text:
This is a
multi-line string.
, Type:

3. Boolean Type: `bool`

Booleans represent truth values: `True` or `False`. They are fundamental for logic and control flow.


is_student = True
is_admin = False

print(f"Is student: {is_student}, Type: {type(is_student)}")
print(f"Is admin: {is_admin}, Type: {type(is_admin)}")
Expected Output: 
Is student: True, Type: 
Is admin: False, Type:

4. Sequence Types: `list`, `tuple`, `range`

These are ordered collections of items.

  • `list` (List): Ordered, mutable (changeable) collection. Defined with square brackets `[...]`.
  • `tuple` (Tuple): Ordered, immutable (unchangeable) collection. Defined with parentheses `(...)`.
  • `range` (Range): An immutable sequence of numbers, often used in loops.

my_list = [1, "apple", True]
my_tuple = (10, 20, 30)
my_range = range(5) # 0, 1, 2, 3, 4

print(f"List: {my_list}, Type: {type(my_list)}")
print(f"Tuple: {my_tuple}, Type: {type(my_tuple)}")
print(f"Range: {list(my_range)}, Type: {type(my_range)}") # Convert range to list for printing
Expected Output: 
List: [1, 'apple', True], Type: 
Tuple: (10, 20, 30), Type: 
Range: [0, 1, 2, 3, 4], Type:

5. Mapping Type: `dict` (Dictionary)

Dictionaries are unordered collections of key-value pairs. Keys must be unique and immutable (like strings or numbers), while values can be of any type.


person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

print(f"Dictionary: {person}, Type: {type(person)}")
print(f"Name: {person['name']}")
Expected Output: 
Dictionary: {'name': 'Alice', 'age': 25, 'city': 'New York'}, Type: 
Name: Alice

6. Set Types: `set`, `frozenset`

Sets are unordered collections of unique items. They are useful for mathematical set operations like union, intersection, etc.

  • `set` (Set): Mutable, unordered collection of unique items. Defined with curly braces `{...}` or `set()`.
  • `frozenset` (Frozenset): Immutable version of a set.

my_set = {1, 2, 3, 2, 1} # Duplicates are automatically removed
my_frozenset = frozenset([4, 5, 6])

print(f"Set: {my_set}, Type: {type(my_set)}")
print(f"Frozenset: {my_frozenset}, Type: {type(my_frozenset)}")
Expected Output: 
Set: {1, 2, 3}, Type: 
Frozenset: frozenset({4, 5, 6}), Type:

Understanding these basic data types is fundamental to writing any Python program. You'll use them constantly to store and manipulate information.
