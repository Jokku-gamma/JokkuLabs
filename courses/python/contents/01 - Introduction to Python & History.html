Introduction to Python & Its History

Welcome to the first lesson of the Python Programming course! In this module, we'll embark on a journey to understand what Python is, why it's so popular, and its fascinating history.

What is Python?

Python is a high-level, interpreted, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python's design philosophy emphasizes code readability with its notable use of significant indentation. It supports multiple programming paradigms, including object-oriented, imperative, and functional programming.

Its extensive standard library and vast ecosystem of third-party libraries make it suitable for a wide range of applications, from web development and data analysis to artificial intelligence and scientific computing.

A Brief History of Python

Python's development began in the late 1980s by Guido van Rossum at CWI in the Netherlands as a successor to the ABC language. The name "Python" itself comes from the British comedy group Monty Python, as Guido was a big fan.

  • 1991: Python 0.9.0 is released.
  • 1994: Python 1.0 is released with new features like lambda, map, filter, and reduce.
  • 2000: Python 2.0 is released, introducing list comprehensions and a garbage collection system.
  • 2008: Python 3.0 (also known as "Python 3000" or "Py3k") is released. This was a major, backward-incompatible revision of the language, designed to fix fundamental flaws and make the language cleaner. Key changes included print becoming a function, improved Unicode support, and changes to integer division.
  • Today: Python 3.x is the current and actively developed version, widely adopted across industries for its simplicity and power.

Python's journey from a hobby project to one of the most widely used programming languages is a testament to its flexibility, readability, and the strong community built around it.

First Python Code Example: "Hello, Jokkulabs!"

Let's write our very first Python program. It's a tradition to start with a "Hello, World!" program. We'll make it "Hello, Jokkulabs!".


# This is a comment in Python
# The 'print()' function is used to display output to the console.
print("Hello, Jokkulabs!")
Expected Output: 
Hello, Jokkulabs!

This simple line of code demonstrates Python's straightforward syntax. You don't need semicolons at the end of lines, and the `print()` function is intuitive.
